# Business Transformation Advisory Website

Premium consulting website for a UAE/GCC business transformation advisory firm focused on AI adoption, Microsoft modernisation, governance, RevOps transformation and SaaS cost control.

## Tech Stack

- Static HTML
- Modular CSS
- Dependency-free JavaScript
- CSS design system with responsive layouts and accessibility states

## Run Locally

```bash
npm run dev
```

You can also open `index.html` directly in a browser.

## Build

```bash
npm run build
```

## Deploy

The project builds to `dist/` and can be deployed to Vercel, Netlify, Cloudflare Pages or any static host.

Suggested settings:

- Build command: `npm run build`
- Output directory: `dist`
- Node version: 18+

## Content Source

Website positioning and copy are based on the attached `Business_Transformation_Advisory_Business_Plan.pdf`, especially the focus on GCC mid-market and regulated organizations, Microsoft Copilot and Azure modernization, governance and compliance, RevOps process transformation, SaaS cost control, and executive advisory.
